# Generator script for browser favicons based on SVG 

## Prerequisites
### MacOS
Install imagemagick with SVG support based on package manager brew

```bash
brew install imagemagick librsvg
```

## Source images
The images *"source/large.svg"* and *"source/small.svg"* will be taken as basis for small and large favicons based on the different device types or resolutions.
